# Restaurant Billing System (Python)

This is a beginner-level Python project that simulates a restaurant ordering system.

- Food and drink menu using dictionaries
- Takes user orders
- Calculates total bill
- Beginner-friendly logic

## Concepts Used
- Python dictionaries
- if-else conditions
- User input
- Basic billing logic
